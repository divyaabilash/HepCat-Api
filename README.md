its the server for the desktop and mobile api. 

to run the project(on mac)
1. in the terminal run <b>mongod</b>
2. open a terminal and run <b> mongo test </b>
3. open the new terminal , navigate to the place where you have downloaded the project and run the command <b> node server.js</b> or to monitor the changes run <b>nodemon server.js</b>
4. open browser and run the localhost:8006
